# Hoshino 夜空 / 织星成座 — 设计 spec

日期：2026-06-13　作者：织星 / Astraea（与 Claude 协作）

## 目标

给 Hoshino 主题加一层"有个人味道"的表达，落点是 **Astraea = 室女座** 的神话身份（Astraea 是化身室女座升天的星之女神，"织星"即把自己织进群星）。把先前讨论的三个方向（彩蛋 / 织星世界观 / 环境氛围）熔成一个连贯的"夜空"，而非三个零散功能。同时解决"现有花瓣不切合主题"——不删花瓣，而是让它融入星空。

## 触发方式（v1）

夜空氛围**绑定到已解析的暗色模式**（resolved `data-color-mode="dark"`）：
- 亮色模式 = 白天：花瓣照常粉色下坠（现状）。
- 暗色模式 = 夜晚：同一套粒子改为向上飘、褪为星屑色；背景浮现室女座。
- 复用现有 Light / Dark / System 三态开关，不新增模式。System 在多数移动端本就随昼夜切换，间接实现"随本地时间"。

后续可选（phase 2，本期不做）：新增第 4 态"跟随星辰"，按访客本地时钟在傍晚自动切到夜晚。

## 四个部件（各自独立、可单独开关）

1. **昼夜天空层** — 由暗色模式驱动；暗色下背景即你现有的深紫夜空，叠加星空氛围。
2. **花瓣进化** — 暗色下粒子反向上升 + 褪色为星屑白/极淡粉。一套粒子两种状态，随模式切换重生成。
3. **本命星图：室女座** — 暗色下背景浮现一幅极淡的室女座连线 + 角宿一(Spica)用樱花粉点亮。纯装饰、固定、置于内容之下（z-index 0，不挡阅读）。
4. **藏起来的星（彩蛋，deferred）** — 室女座中一颗隐藏星，点亮后连出一句作者手写的话；夜空中鼠标划过带起星屑。依赖夜空已稳定，故二期实现，文案由作者提供。

## 约束 / 取舍

- 全部尊重 `prefers-reduced-motion`：关动效者只看到静态夜色，不飘、不连线、不闪烁。
- 雅致优先：星点密度、连线透明度、夜色深浅都要克制，避免花哨。亮度/透明度上限在实现时反复目测调。
- 不与现有 `.petals` 粒子层、`body::before` 网点底纹、自定义光标打架；夜空层独立、z-index 在内容之下。
- 单主色纪律延续：夜空里唯一的强调色仍是樱花粉（Spica），其余星点走中性白/淡紫。

## 实施顺序

①③ 夜空 + 星座 → ② 花瓣进化 → 各自在 preview 的暗色模式下目测验证 → ④ 彩蛋（待作者给文案）。
